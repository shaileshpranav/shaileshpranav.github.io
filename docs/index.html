Projects

  • Interactive AI Chatbot for Enhanced User Engagement

    The Interactive AI Chatbot for Enhanced User Engagement project developed a sophisticated chatbot using the Mixtral 8x7b / Llama 3 7b model to drive engaging conversations with advanced NLP capabilities. A user-friendly interface was created with Streamlit, ensuring seamless interaction. The project utilized Neo4j for knowledge graph-based storage, Milvus for vector storage, and Langchain for Retrieval Augmented Generation, facilitating accurate and relevant responses. The chatbot can be securely deployed on-premises or in the cloud, enhancing user experience and engagement through precise and contextually appropriate answers. This integration sets a new standard for digital interactions.
